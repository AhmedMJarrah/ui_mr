# ⚖️ منظومة مراجعة التشريعات

نظام مراجعة وتدقيق الجهات والأنواع المرتبطة بالقوانين والأنظمة.

## المميزات

- رفع ملفات CSV أو Excel
- مراجعة كل سجل بشكل منفرد (الجهة + النوع)
- تعديل البيانات يدوياً
- تتبع حالة كل سجل (لم يُراجع / معتمد / معدّل)
- شريط تقدم وإحصائيات لحظية
- تنزيل الملف المعدّل بنفس الصيغة الأصلية مع أعمدة مراجعة جديدة

## الأعمدة المطلوبة في الملف

| العمود | الوصف |
|--------|--------|
| `leg_name` | اسم القانون / النظام |
| `entity_final` | الجهة المعنية |
| `type` | نوع الجهة |

## التشغيل المحلي

```bash
pip install -r requirements.txt
streamlit run app.py
```

## النشر على Streamlit Cloud

1. ارفع المجلد إلى GitHub
2. اذهب إلى [share.streamlit.io](https://share.streamlit.io)
3. اختر الـ repo وحدد `app.py` كملف رئيسي
4. انقر **Deploy**

## الأعمدة المضافة في ملف التصدير

| العمود | الوصف |
|--------|--------|
| `entity_audited` | الجهة بعد المراجعة |
| `type_audited` | النوع بعد المراجعة |
| `audit_status` | الحالة: لم يُراجع / معتمد / معدّل |
| `audit_notes` | ملاحظات المراجع |
